Favourite Tweets by Twitter Inc.

Added March 16th, 2015

Description

Here's a quick overiew of what this integration will allow you to do.

Twitter is a social network and microblogging service where users post 140-character messages called "tweets" to their followers.

Favourite Tweets will allows you to save any tweet you favourite, which has a URL to your Wunnderlist.

Authentication

Give Wunderlist access to your Twitter account. Access can be revoked at any time.

Connected to Twitter as @jarred

Choose List

Choose the list you'd like Twitter to be connceted to.

We've suggested a list but you can choose another one or create a new one.

 
Save Settings Delete Integration

Deleting this integration is permenant and can not be CTRL + Z'ed.
